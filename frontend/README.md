# AI Auditing System - Frontend

Modern Next.js frontend for the AI Auditing System with shadcn/ui components.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Beautiful Landing Page**: Modern hero section with feature highlights
- **Dashboard**: View and filter all audits with real-time updates
- **Upload Page**: Upload documents to start new audits
- **Review Page**: Detailed audit review with compliance flags and statistics

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Radix UI primitives

## API Integration

The frontend connects to the Flask backend API running on `http://localhost:5000`. Make sure the backend is running before using the frontend.
